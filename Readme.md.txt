Stage-1:
This applications lets user know the current movies playing in theaters, sort movies by popularity-rating.

How to use
1.When presented with movies click on any one and you will be directed to details about the movie.
2.You can sort movies by selecting the filter option.

NOTE: In the application api_key should be unique for every individual, you can get one by creating an account with theMoviedb.org and requesting them for one.
